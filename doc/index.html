docist.generator

:added: 0.1 :author: Chad Angelelli

Generate static documentation from source files.

Output formats:

  1. :edn (default): EDN map
  2. :json: JSON string
  3. :yaml: YAML string

> NOTE: Output is processed by default via categorize-parsed. Provide :uncategorized? option to disable.

Defaults

  • :output-dir: "doc"
  • :output-format: :edn
